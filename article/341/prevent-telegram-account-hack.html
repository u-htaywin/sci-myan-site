အကယ်လို့ ဖုန်း အော်ပရေတာကို မထိန်းချုပ် နိုင်ဘူး ဆိုရင်တောင် လေလှိုင်းထဲမှာ သွားနေတဲ့ ဒီ SMS တွေကို ကြားဖြတ်ဖမ်းယူနိုင်တဲ့ နည်းလမ်းတွေ ရှိပါသေးတယ်။ အထူသဖြင့် hacker ဂုဏ်းကြီးတွေဆိုရင် ကြားဖြတ်ဖမ်းယူ နိုင်ပါတယ်။

2FA ကို ဘယ်လို ဖွင့်ရမလဲ

2FA ဖွင့်တာက လွယ်ပါတယ်။

  1. ကိုယ့်ဖုန်းပေါ်က Telegram App အောက်ခြေက ကိုယ့်ပုံကို ထောက်လိုက်ပြီး Settings ထဲ ပြန်ဝင်ပါ။
  2. Privacy and Security ကို ထောက်လိုက်ပါ။ Privacy and Security Setting ထဲ ရောက်သွားပါမယ်။


  3. Two-Step Verification ကို ထောက်ပါ။


  4. နောက် screen မှာ Set Additional Password ဆိုတာကို ထောက်ပါ။


  5. နောက် screen မှာ Password ရိုက်ထည့်ပါ။ နှစ်ခါ ရိုက်ထည့်ပါမယ်။

  6. Password မေ့သွားရင် သတိရအောင် Password Hint လေး ထည့်ပါ။


  7. Password လုံးဝ မေ့သွားရင် ပြန် reset လုပ်ရအောင် ကိုယ့်မှာ email ရှိရင် recovery email အနေနဲ့ ထည့်ထားလို့ ရပါတယ်။ မရှိလဲ ကိစ္စ မရှိပါဘူး။ PW သာ မမေ့အောင် မှတ်ထားလိုက်ပါ။ ​Recovery email မရှိရင် အောက်ဆုံးက Skip setting email ကိုထောက်ပြီး ဆက်သွားပါ။
  8. Recovery email ကို ခုနက အဆင့်မှာ ထည့်ခဲ့ရင် code နံပါတ် လှမ်းပို့ပေးပါလိမ့်မယ်။ အဲ့သည် ကုဒ်နံပါတ်ကို နောက်တဆင့်မှာ ရိုက်ထည့်ရပါမယ်။ Recovery email set မလုပ်ခဲ့ဘူးဆိုရင်တော့ ဒီအဆင့် မပါတော့ပါဘူး။
  9. Return to Setting ကို နှိပ်ပြီး Setting စာမျက်နှာကို ပြန်သွားပါ။
  10. ပိုသေချာအောင် Privacy and Security Setting ထဲပြန်ထွက်ပြီးရင် Phone Number ကို ထောက်လိုက်ပါ။
